# grappy
Goodreads profile scraper using scrapy.
Python version 2.7

## installation

```
pip install Scrapy
git clone https://github.com/andrei8055/grappy.git
cd grappy
vim config.ini //set username, password, start point profile
```

## usage
Scrap profiles
```
scrapy crawl grappy 
```

Display profile details to console
```
python db.py
```

...or use `dbeaver`
